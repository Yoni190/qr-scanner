# QR Scanner

A simple Windows desktop program that scans QR codes.

---

## Download and Run

Get the latest Windows executable from the [Releases](https://github.com/Yoni190/qr-scanner/releases/tag/v1.1).  

1. Download the `.exe` file.  
2. Double-click it to run.
3. Make sure your webcam is on. 

>Press **Q** to close the application.

---

## Running Locally from Source

If you want to run the program from source instead of the `.exe`, follow these steps:

### 1. Install dependencies

```bash
pip install opencv-contrib-python
```

### 2. Clone repository

```bash
git clone https://github.com/Yoni190/qr-scanner
cd qr-scanner/
```

### 3. Run the python file

```bash
 python scanner.py
```
